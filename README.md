# LagrangianTraj_MF

### Authors: 
Meryl WIMMER, Philippe ARBOGAST, Vinita DESHMUKH, Léo DUCONGE, Gwendal RIVIERE, Sébastien FROMANG, Nicolas CHIABRANDO, Carole LABADIE

### Description: 
Scripts to compute Lagrangian trajectories for different models

## How to use:

### 1) Download the git repository: 

git clone https://github.com/meryl-wimmer/LagrangianTraj_MF.git

### 2) Create and use a python virutal environnement (named MyVenv) in Jupyter Notebook: 

python -m venv MyVenv 

pip3 install --user ipykernel

python -m ipykernel install --user --name=MyVenv

### 3) Open Jupyter Notebook :

jupyter-notebook &

### 4) To run each block :

Click on the play button
